# Weekly Review Checklist

> Your 30-minute weekly procedure. Do this every Sunday morning.

---

## Checklist

### 1. Read the Telegram Digest (5 min)

- [ ] Open Telegram → find the weekly portfolio summary
- [ ] Note any RED health scores (below 40)
- [ ] Note total revenue for the week
- [ ] Note any error alerts from the week

### 2. Review Opportunities (5 min)

- [ ] Read the weekly opportunity digest (top 5 from Daily Researcher)
- [ ] For each: mark as "pursue" / "watch" / "skip" via Telegram reply
- [ ] If "pursue" — add a note to [[Opportunity Log]] with what to build

### 3. Approve Capital Moves (5 min)

- [ ] Check Telegram decision queue for pending approvals
- [ ] Review any capital reallocation suggestions
- [ ] Approve or modify via Telegram reply
- [ ] Reminder: No single strategy should get >40% of capital

### 4. Update the Vault (10 min)

- [ ] Update [[Dashboard]] strategy status table
- [ ] Update [[Revenue Tracker]] with this week's numbers
- [ ] Update each active strategy note's Metrics Log table
- [ ] Log any decisions in [[Decision Log]]
- [ ] Check [[Performance Log]] for any error patterns

### 5. Quick Health Check (5 min)

- [ ] Is n8n running? (Check Telegram heartbeat received today)
- [ ] Any workflows failed this week? (Check Performance Log)
- [ ] Are trading bots within risk limits? (Check drawdown)
- [ ] Are API costs within budget? (Check Revenue Tracker costs)

---

## Monthly Additions (First Sunday of Month)

- [ ] Update monthly summary in [[Revenue Tracker]]
- [ ] Calculate overall portfolio ROI
- [ ] Review partner cost-sharing if applicable ([[Partner - Portfolio]])
- [ ] Consider: Should any strategy be killed? (Score <20 for 2+ weeks)
- [ ] Consider: Should any honorable mention strategy be added?
- [ ] Update [[Dashboard]] Recent Activity section

---

## Quarterly Review (Month 3 and Month 6)

- [ ] Full portfolio valuation (revenue + asset values)
- [ ] Compare actual vs projected in each strategy note
- [ ] Review all [[Decision Log]] entries — were decisions right?
- [ ] Is the adaptive system (Daily Researcher) finding useful opportunities?
- [ ] Should we add more capital from revenue?
- [ ] Update probability distribution estimates based on actual data
